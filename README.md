`Planara.Common` — общий пакет с инфраструктурными примитивами и контрактами,
используемыми во всех backend-сервисах.

Пакет не содержит бизнес-логики и предназначен для повторного использования
в микросервисах и API.

---

## Установка

```bash
dotnet add package Planara.Common
```
